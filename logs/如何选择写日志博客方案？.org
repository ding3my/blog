** CSDN等第三方平台
缺点：
- 无法版本控制（即无法备份）

** Github Issues
缺点：
- 无法版本控制（即无法备份）

** GitHub Pages & Markdown
缺点：
- Emacs Markdown插件操控性不如Emacs Org

** GitHub Pages & Org
优点：
- 支持版本控制（即支持备份）
- Emacs org功能强大，文档完整。

缺点：
- Org通用性不如Markdown，例如bitbucket、gitee（osc码云）、CSDN等不支持。但是可以通过导出为html文件来解决（html更通用，只是需要静态服务器支持）。
